# Tutorial para visualização de dados no Datawrapper

<p> Nesse tutorial, será mostrado o passo a passo para a criação de um gráfico simples na plataforma <a href="https://www.datawrapper.de/"> Datawrapper</a>, que permite que o usuário produza mapas, gráficos e tabelas personalizados e compatíveis para serem publicados em sites e páginas da web.
